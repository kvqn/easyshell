# Debugging Tips for Shell Scripts

Debugging isn't scary; it's just problem-solving with extra steps.\
Here’s how to fix those annoying errors in your scripts.

---

## Steps to Debug

1. Add `set -e` to stop the script when errors occur.
1. Use `echo` generously—print variables, commands, and results.
1. Check file paths and permissions—they’re often the culprits.

Mantra: _"Every error is an opportunity to learn."_
