# Bash Tricks for Cool Coders

Bash isn't just a shell; it's a playground for creative coding.\
Learn these tricks to impress your friends (and yourself).

---

## Bash Magic

- Use `alias` to create shortcuts for your favorite commands.
- Use `$()` for command substitution—it’s like magic in parentheses.
- Use loops (`for`, `while`) to automate repetitive tasks.

Pro Tip: Combine loops with conditionals (`if`, `else`) for ultimate scripting power.
