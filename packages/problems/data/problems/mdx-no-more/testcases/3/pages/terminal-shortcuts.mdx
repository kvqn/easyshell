# Terminal Shortcuts You Need to Know

Save time and type less with these handy shortcuts.

---

## Must-Know Shortcuts

- **Ctrl + C**: Stop whatever’s running (your panic button).
- **Ctrl + Z**: Pause a process and send it to the background.
- **Ctrl + R**: Search through command history like a time traveler.
- **Tab**: Auto-complete commands or file names—because typing is hard.

Pro Tip: Mastering shortcuts makes you look cooler than any GUI user.
